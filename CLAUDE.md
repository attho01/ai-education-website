# 나의 행복한 직업 찾기!!

## 프로젝트 설명
직업상담을 통해 행복한 직업을 찾아주는 웹페이지입니다.
사용자의 관심사와 성향을 파악하여 적합한 직업을 추천합니다.

## 사용 기술
- **HTML**: 웹페이지 구조 작성
- **CSS**: 웹페이지 디자인 및 스타일링
- **JavaScript**: 상호작용 기능 및 직업 추천 로직

## 파일 구조
```
프로젝트 루트/
├── index.html          # 메인 페이지
├── style.css           # 스타일시트
├── script.js           # JavaScript 로직
└── CLAUDE.md           # 프로젝트 설명 문서
```

## 실행 방법
1. 프로젝트 폴더를 엽니다
2. `index.html` 파일을 웹 브라우저(Chrome, Firefox, Edge 등)에서 엽니다
3. 직업 상담 질문에 답하고 결과를 확인합니다

## 코딩 규칙
### 주석 작성
- **모든 주석은 한국어로 작성**합니다
- 코드의 목적과 기능을 명확히 설명합니다

### 코드 작성 원칙
- **초보자가 이해하기 쉽게** 작성합니다
- 복잡한 로직은 간단한 단계로 나누어 설명합니다
- **모든 함수와 중요한 코드 블록에 설명 주석을 추가**합니다
- 변수명과 함수명은 의미를 명확히 알 수 있도록 작성합니다

### 주석 예시
```javascript
// 사용자의 답변을 저장하는 배열
let userAnswers = [];

// 직업 추천 결과를 계산하는 함수
function calculateCareer() {
    // 답변 점수를 합산
    let score = 0;

    // 각 답변을 순회하며 점수 계산
    for (let answer of userAnswers) {
        score += answer.value;
    }

    return score;
}
```

## 개발 가이드
- 이 프로젝트는 초보자를 위한 학습 프로젝트입니다
- 코드를 수정할 때는 반드시 주석을 함께 업데이트합니다
- 새로운 기능 추가 시 기존 코딩 규칙을 따릅니다
